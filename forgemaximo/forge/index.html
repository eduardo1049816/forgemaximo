Qual é a prioridade de performance?

Isso define o equilíbrio entre CPU e GPU na build.

Máxima GPU
Render em tempo real, CUDA, gaming ultra. A GPU é a estrela.
Máxima CPU
Compilação, simulação, múltiplos cores. CPU lidera.
Equilibrado
Mix de tarefas. Nenhum componente é gargalo.
Alta RAM + Storage
Muitos processos abertos, arquivos grandes, edição RAW.
